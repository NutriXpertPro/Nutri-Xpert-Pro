**Como usar no Tailwind CSS:**
Para aplicar essas cores, use o prefixo da classe seguido do nome da cor. Por exemplo:
*   `bg-atlantis` para fundo
*   `text-denim` para texto
*   `border-shuttle-gray` para borda
